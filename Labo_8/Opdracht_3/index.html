Opdracht 3

Waarom staat de tekst in de blokken hoizontaal gecentreerd?

Tekst staat gecentreerd door de parameter div {text-align: center; }

Waarom heeft de property vertical-align:center geen effect?

Deze property werkt alleen op inline-block, inline en de tabel-elementen

Wat doet vertical-align dan wel?

Het bepaalt de uitling van de inline-elementen tegenover de andere inline-elementen

Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren?

Dit kan je doen met .area { line-height: 100px text-align: center; }

Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren?

Met het volgende element: .area { display: table-cell; vertical-align: middle; text-align: center; }

Waarom krijgt de regel voor class 'tall' voorrang?

Tall is speciefker dus deze krijgt dan meer voorang.
